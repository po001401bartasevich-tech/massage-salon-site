# massage-salon-site
Website for massage salon (lab project)
## Описание проекта
Сайт массажного салона «MassageRelax» предназначен для привлечения клиентов и автоматизации записи на услуги. Посетители могут ознакомиться с каталогом массажных услуг, прочитать отзывы, записаться онлайн, а также зарегистрироваться и отслеживать свои записи в личном кабинете. Администратор салона управляет услугами, мастерами и бронированиями через специальную панель.

## Целевая аудитория
- Взрослые люди (25–60 лет), следящие за здоровьем, желающие снять стресс или пройти лечебный массаж.
- Администраторы салона, которым нужно управлять расписанием и услугами.

## Ключевой функционал
- Просмотр каталога услуг с фильтрацией по типу массажа.
- Детальная страница каждой услуги с описанием, ценой и кнопкой записи.
- Регистрация, вход и восстановление пароля.
- Личный кабинет клиента с историей записей и возможностью отмены.
- Онлайн-запись на выбранную дату и время.
- Система отзывов (только для авторизованных пользователей).
- Поиск по услугам.
- Административная панель для управления контентом.
- Контактная информация и карта проезда.

## Структура приложения (минимум 8 разделов)

| № | Раздел | URL | Тип доступа |
|---|--------|-----|-------------|
| 1 | Главная | `/` | публичный |
| 2 | Аутентификация (вход, регистрация, восстановление пароля) | `/auth/*` | публичный |
| 3 | Каталог услуг | `/services` | публичный |
| 4 | Детальная страница услуги | `/services/:id` | публичный |
| 5 | Личный кабинет клиента | `/profile` | приватный |
| 6 | Онлайн-запись | `/booking` | приватный |
| 7 | Отзывы | `/reviews` | публичный (добавление — приватное) |
| 8 | Контакты | `/contacts` | публичный |
| 9 | Поиск | `/search` | публичный |
| 10 | Админ-панель | `/admin` | приватный (только для администратора) |


## Технологический стек

### Frontend
- **Язык**: HTML5, CSS3, JavaScript 
- **Фреймворк**: не используется (чистый JS) для лучшего понимания основ
- **Роутинг**: History API или хэш-роутинг
- **Управление состоянием**: локальные переменные, localStorage для хранения токена и данных пользователя


### Backend
- **Платформа**: Node.js
- **Фреймворк**: Express
- **API**: REST


### База данных
- **СУБД**: MySQL


### Инфраструктура (хостинг)
- **Frontend**: GitHub Pages — статический хостинг, бесплатный и простой.
- **Backend**: Railway.app — деплой Node.js приложений с бесплатным тарифом.
- **База данных**:  MySQL — облачный кластер с бесплатным уровнем.
